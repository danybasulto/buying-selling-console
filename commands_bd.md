# Comandos XAMPP

## Iniciar base de datos
mysql -h localhost -u root -p

## crear base de datos
create database **demo**;

## Ver bases de datos
show databases;

## Editar base de datos
use **demo**;

### Crear tabla
create table **cliente**(
    -> *codigo_cliente*
    -> *nombre*
    -> );

### Ver tablas
show tables;

#### Mostrar tabla
Describe **cliente**;
